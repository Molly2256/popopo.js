## 🔍 Troubleshooting

### The app does not open

- Try running it again as admin
- Make sure the download finished
- Check that Windows did not block the file
- Move the app to a simple folder like `Downloads` or `Desktop`

### Windows shows a security prompt

- Choose More info if needed
- Then choose Run anyway only if you trust the source
- Make sure you downloaded it from the link above

### The app opens and closes right away

- Restart your PC
- Try opening the app from its folder
- Check that no files are missing from the download

### The screen looks too large or too small

- Use the display settings in Windows
- Change screen scaling
- Open the app again after the change
